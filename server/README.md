# Checkout faster with Link

A [Sinatra](http://sinatrarb.com/) implementation.

## Requirements

- Ruby v2.4.5+
- [Configured .env file](../README.md)

## How to run

1. Install dependencies

```
bundle install
```

2. Run the application

```
ruby server.rb
```

3. Go to `http://localhost:4242` in your browser to see the demo
